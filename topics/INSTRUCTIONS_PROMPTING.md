# Sujet 04 : Prompt Engineering Avancé

## Rappel du sujet

Le **prompt engineering** consiste à concevoir des prompts efficaces pour obtenir les meilleurs résultats des LLMs. Les techniques avancées incluent : few-shot learning, chain-of-thought, templates, optimisation du contexte.

## Tronc commun (obligatoire)

1. **Définir une tâche** : Classification, génération, extraction, etc.
2. **Créer des templates de prompts** : Structure réutilisable
3. **Implémenter few-shot learning** : Exemples dans le prompt
4. **Tester plusieurs variantes** : Comparer l'efficacité
5. **Baseline simple** : Prompt naïf vs optimisé

## Extensions possibles

1. **Chain-of-Thought** : Raisonnement étape par étape
2. **Évaluation systématique** : Tester N variantes automatiquement
3. **Optimisation du contexte** : Réduire les tokens intelligemment
4. **Meta-prompting** : Le LLM génère ses propres prompts

## Pipeline typique

1. Choix de la tâche
2. Prompt initial (baseline)
3. Optimisation progressive
4. Évaluation quantitative
5. Analyse des résultats

## Datasets suggérés

- Tasks de classification
- Q&A
- Génération de résumés
- Extraction d'informations
